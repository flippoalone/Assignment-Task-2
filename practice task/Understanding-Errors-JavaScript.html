1.What is an error in JavaScript?

An Error is an event that disrupts the normal flow of your program. It essentially means something went wrong that prevents the computer from executing the code successfully.

Explain why errors are important

Protection: They stop a program from running with bad data, which could corrupt a database or crash a server.
Debugging: The error message acts like a map. It tells you exactly what went wrong and where (which line number), helping you fix bugs faster.
User Experience: By catching errors, you can show users helpful messages (like "Server is busy, try again") instead of a blank white screen.
